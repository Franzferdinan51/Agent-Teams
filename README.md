## рҹ•ёпёҸ P2P DECENTRALIZED MESH

**No central server вҖ” pure peer-to-peer agent communication.**

Inspired by [BitChat](https://github.com/permissionlesstech/bitchat).

```
      рҹ•ёпёҸ Agent A в—„в”Җв”Җв”Җв”Җв”Җв”Җв–ә рҹ•ёпёҸ Agent B
         в”Ӯ    \            /    в”Ӯ
         в”Ӯ     \          /     в”Ӯ
         в”Ӯ      \        /      в”Ӯ
         в”Ӯ    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”җ   в”Ӯ
         в”Ӯ    в”Ӯ  рҹӣ°пёҸ Relay   в”Ӯ   в”Ӯ (optional)
         в”Ӯ    в””в”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ   в”Ӯ
         в”Ӯ           в”Ӯ           в”Ӯ
         в–ј           в–ј           в–ј
      рҹ•ёпёҸ Agent C в—„в”Җв”Җв”Җв”Җв”Җв”Җв–ә рҹ•ёпёҸ Agent D
```

- **Gossip protocol** вҖ” messages spread node-to-node
- **TTL limits** вҖ” prevents infinite loops
- **Optional relays** вҖ” helps nodes behind NAT
- **No blockchain** вҖ” pure P2P networking

```bash
# Start relay (optional)
./scripts/hive-p2p.sh 4100 relay

# Start peer
./scripts/hive-p2p.sh 4101 peer

# Works over Tailscale VPN too!
```
