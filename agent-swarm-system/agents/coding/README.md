## Model Assignments

| Tier | Model | Use Case |
|------|-------|----------|
| Tier 1 | `bailian/qwen3.5-plus` | Strategic decisions, architecture, complex reasoning |
| Tier 2 | `bailian/glm-5` | Framework expertise, coding, mid-complexity |
| Tier 3 | `bailian/MiniMax-M2.5` | Quick tasks, scripting, documentation |

Each agent file contains:
- Role definition with YAML frontmatter
- Responsibilities and deliverables
- Quality focus areas
- Output format specification
- Escalation path
