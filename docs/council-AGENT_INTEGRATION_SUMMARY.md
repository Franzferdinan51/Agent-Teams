## 🚀 How to Use

### OpenClaw Integration

#### Via MCP:
```bash
# AI Council is auto-connected via MCP
# Just start using it in OpenClaw!
```

#### Via Skills:
```bash
# Start deliberation
openclaw agent --message "council deliberate Should we implement feature X?"

# Check status
openclaw agent --message "council status"

# Export session
openclaw agent --message "council export session_123"

# List sessions
openclaw agent --message "council list sessions"

# Join session
openclaw agent --message "council join session_123"
```

#### Via Web UI:
```
Open http://localhost:3003/
All OpenClaw integrations are active!
```

### Multi-Agent Collaboration

#### Via Agent Registry:
```bash
# Discover agents
curl http://localhost:3001/api/agents

# Register agent
curl -X POST http://localhost:3001/api/agents \
  -H "Content-Type: application/json" \
  -d '{"name": "my-agent", "capabilities": ["research", "analysis"]}'

# Check agent health
curl http://localhost:3001/api/agents/my-agent/health
```

#### Via Task Delegation:
```bash
# Delegate task
curl -X POST http://localhost:3001/api/tasks/delegate \
  -H "Content-Type: application/json" \
  -d '{
    "task": "Research best practices",
    "agent": "research-agent",
    "priority": "high"
  }'

# Check task status
curl http://localhost:3001/api/tasks/task_123/status
```

### Cross-Platform Integration

#### Via Agent Gateway:
```bash
# View all agents
curl http://localhost:3001/api/gateway/agents

# Trigger agent action
curl -X POST http://localhost:3001/api/gateway/agents/langchain/action \
  -H "Content-Type: application/json" \
  -d '{"action": "search", "query": "AI best practices"}'

# View analytics
curl http://localhost:3001/api/gateway/analytics
```
